# AI Core Service

Nucleul AI pentru sistemul Ahauros.

## Funcționalități

- Modele de machine learning
- Procesare NLP
- Computer vision
- API pentru servicii AI

## Tehnologii

- Python
- TensorFlow / PyTorch
- FastAPI
- Docker

## Dezvoltare

```bash
# Instalare dependențe
pip install -r requirements.txt

# Pornire serviciu
uvicorn app:app --reload
```
