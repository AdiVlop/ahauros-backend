# Decision Engine Service

Motor de decizii AI pentru sistemul Ahauros.

## Funcționalități

- Procesare modele AI
- Algoritmi de decizie
- Integrare cu servicii externe
- Analiză predictivă

## Tehnologii

- Python / TensorFlow / PyTorch
- FastAPI / Flask
- Redis
- ClickHouse

## Dezvoltare

```bash
# Instalare dependențe
pip install -r requirements.txt

# Pornire serviciu
python app.py
```
